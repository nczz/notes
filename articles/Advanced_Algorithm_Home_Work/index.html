主題

  1. 作業一： 找一個「In place」的演算法問題？

    答： 數值比 大 / 小，程式碼如下 (JavaScript)

     var arr = [10,8,5,4,6,3,2,1,9,7];
 var min = arr[0];
 for (var i=1; i<arr.length; ++i) {
     if (arr[i] < min) {
         min = arr[i];
     }
 }
 console.log("The minimum in arr is: "+min);

  2. 作業二： 證明「選擇、泡沫、插入排序法是否屬於 Stable / Non-Stable」？

    答：維基說原始的選擇排序法為 Non-Stable 的排序法！但結果其實取決於程式的寫法，在判別大小時如果多一個 = 號就會是不穩定的排序了。

    選擇排序法： 在陣列中找到最小的後放在最左邊設定其為排好的數列，再從下一個開始到結尾找最小的排，直到結束。

     var arr = genRandNumArr(10); //產生陣列
 var work_index = 0 //設定起始位址
 var min = {index:work_index,value:arr[work_index].value}; //設定最小值，即為起始位址
 for (var j=0; j<arr.length; ++j) {
     for (var i=work_index; i<arr.length; ++i) {
         if (arr[i].value < min.value) { //non-stable switch
             min.value = arr[i].value; //紀錄最小值
             min.index = i; //紀錄最小值位址
         }
     }
     arr.swap(work_index,min.index); //將最小的元素換到最左後從下一個繼續比
     work_index = j;
     min.index = work_index;
     min.value = arr[work_index].value;
 }

    泡沫排序法： 在陣列中兩兩比較，找最大的數一路交換到最右邊，重複比較步驟直到換到最左。

     for (var r=arr.length-1; r>=0; --r) {
     for (var i=0; i<r; ++i) {
         if (arr[i].value > arr[i+1].value) { //non-stable switch
             arr.swap(i,i+1);
         }
     }
 }

    插入排序法： 設定從第二個元素開始到結束，比較迴圈過程中指定的元素前面所有元素大小，交換比指定元素大的元素，直到迴圈結束。

     for (j=1;j<arr.length ;j++ ) {
     for (var t=j-1; t>=0; --t) {
         if (arr[t].value > arr[t+1].value) { //non-stable switch
             arr.swap(t,t+1);
         }
     }
 }
