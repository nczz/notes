HTML 樣板引擎 - Jade

Written by 莫小屁

前言

第一次接觸這語言是在實做 NodeJS + ExpressJS 碰到的，App 中預設的樣版引擎模組， 當時沒覺得甚麼，一直到開始想改這 Blog 的模板時就再次相遇。

主題

官網： Jade - Node Template Engine 、 Github

對常寫 HTML 的人應該很好上手，語言的表達很清楚，唯一要注意的大概就是排版吧!

用 Tab 排或用 Space 空格排都可以，但要維持一致性，不然就會發生解析錯誤、編譯失敗。

舉簡單例子：

一般寫 HTML 標籤都是帶著角括弧成對出現，在 Jade 中只有一半的標籤而且無需角括弧，然後利用縮排的觀念去解釋

EX

<html>
    <head>
        <title>Hi, This is Mxp.</title>
    </head>
    <body>
        <p>Hello World!</p>
    </body>
</html>

轉成 Jade 後變成

html
  head
    title
      Hi, This is Mxp.
    body
      p
        Hello World!

看起來好像沒這麼漂亮，但是實用性大增，程式化之下可以做的事就更簡單！

像這個 Blog 每一頁面都是靜態的 html 檔，由 Jade 樣板產生的話只需要幾個檔案搭配一些繼承關系就解決，

有規劃的話，對於開發效率應該提升不少。

以下列出幾個網路上學習的資源，看過玩過就有收穫了！

  1. Learnjade.com — 建議初學者從這開始，一步一步學，邊學邊改
  2. Playground — 與上一個網站相同，但要特別推薦這個是因為可以在這邊模擬繼承關係
  3. Jade-syntax-docs — 與第一個性質學習差不多，但有多提到了函式的操作與一些基礎概念
